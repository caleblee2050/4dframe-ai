# 4DFrame AI 짝코더 — 페이지 안내

배포 URL: **https://4dframe-ai.vercel.app**

학생/교사/관리자 누구에게든 그대로 전달 가능한 형태로 정리.

---

## 학생용 (메인)

### 🌟 쉬운 모드
**https://4dframe-ai.vercel.app/play/simple**

학생이 직접 사용하는 메인 페이지입니다. THINK GEN 디자인이 적용된 단순 UI 로, 작품 선택 → 자연어 입력(타자/마이크) → 자동 실행 흐름. 다국어(한국어/English/Монгол) 지원, 거리센서 토글, 자동차 작품 시 조이스틱 자동 노출, 악어 작품 시 손 펼침/주먹으로 입 조작 가능.

핵심 기능:
- 작품 선택 (자유/바이킹/자동차/회전그네/악어/발레리나)
- 자연어 입력 + 마이크 (한/영/몽 자동 인식)
- AI 즉석 작곡 — "긴장되는 음악" 같은 분위기 묘사도 음악으로
- 1/2/3 빠른 실행 칩 — 어른이 고급 모드에서 저장한 동작 자동 노출
- 입 벌리기/다물기 직접 버튼 (악어 서보)
- 거리센서 ON/OFF 토글 (AI 가 거리에 반응)
- 부모용 ⚙️ 설정 모달 (모터/서보 캘리브)

## 부모/교사용

### ⚙️ 고급 모드
**https://4dframe-ai.vercel.app/play**

모든 기능이 노출된 어드밴스드 페이지입니다. 자연어로 program 만들어 💾 저장하면 학생의 쉬운 모드 칩으로 자동 반영. 모터 캘리브, 서보 각도, 손동작 매핑, 카메라 제스처, 직접 명령 송신 등 미세 조정 가능.

핵심 기능:
- 모든 자연어 + 음성 + 카메라 + 조이스틱
- 모터 길들이기 (시동 V 조절, 정/역 토글, 시동 테스트)
- 서보 각도 캘리브 (SA 입 / SB 꼬리 ±15°)
- 손동작 매핑 (손가락 0~5개 → 동작 6슬롯 직접 지정)
- 직접 조종 (조이스틱 + 키보드 화살표)
- 거리 반응 모드
- 💾 program 저장 → 쉬운 모드에 자동 노출

## 권장 시연 흐름

1. **학생에게**: `https://4dframe-ai.vercel.app/play/simple` 만 알려주면 됨.
2. **시연 전 준비** (어른이): `/play` 에서 자연어로 program 만들고 💾 저장 → 학생 화면에 자동 칩 등장.
3. **학생 흐름**:
   - 작품 선택 (예: 악어)
   - 1/2/3 칩 클릭 → 즉시 실행 (음악 + 동작 sync)
   - 또는 자연어 직접 입력 (예: "두근두근 심장박동에 맞춰 입 벌렸다 다물어줘")
   - 마이크 🎤 — 말하면 자동 변환
   - 손 동작 (악어): 손 펼침 = 입 벌림 / 주먹 = 입 다묾
4. **시연 중 부모/교사**:
   - 우측 하단 ⚙️ 클릭 → 모터 길들이기, 서보 각도, 거리 반응
5. **언어 변경**: 우측 상단 🇰🇷 🇺🇸 🇲🇳 클릭 → UI + AI 응답 + TTS voice 즉시 전환
